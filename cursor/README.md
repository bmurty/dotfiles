# Cursor Configuration

Here are the contents of my [Cursor](https://cursor.com/download) configuration directory.

## Usage

1. Download and install the [CommitMono](https://commitmono.com/) custom font
2. Install [Cursor](https://cursor.com/download), open it and wait for it to load
4. Close Cursor so that the settings files are initialised
5. Open Terminal
8. Copy [settings.json](settings.json) from this directory to your [Cursor Configuration Directory](https://cursor.com/docs/cli/reference/configuration#file-location)
9. Open Cursor
