# Zed Editor Configuration

Here are my settings for [Zed Editor](https://zed.dev/).

1. Download and install the [CommitMono](https://commitmono.com/) custom font
2. Install [Zed Editor](https://zed.dev/), open it, wait for it to load, then close it
3. Copy the contents of [settings.json](settings.json) to your [User Settings File](https://zed.dev/docs/configuring-zed#user-settings-file)
4. Copy the contents of [keymap.json](keymap.json) to your [User Keymaps File](https://zed.dev/docs/key-bindings?highlight=keymap#user-keymaps)
5. Open Zed Editor again

For more information about the available configuration options for these files, refer to the [Configuring Zed](https://zed.dev/docs/configuring-zed) and [User Keymaps](https://zed.dev/docs/key-bindings) pages.
