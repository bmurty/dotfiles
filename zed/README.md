# Zed Editor Configuration

Here are my settings for [Zed Editor](https://zed.dev/).

1. Download and install the [CommitMono](https://commitmono.com/) custom font
2. Install [Zed Editor](https://zed.dev/), open it and wait for it to load
3. Copy the contents of [settings.json](settings.json) from this directory to your [Zed User Settings File](https://zed.dev/docs/configuring-zed#user-settings-file)
4. Open Zed

For more information about the available configuration options, refer to the [Configuring Zed page](https://zed.dev/docs/configuring-zed).

To see all available settings, open the Command Palette, then run `zed: open default settings`.
